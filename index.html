MSF - Pandemic and Epidemic

The proyect's goal was to promote the work of MSF worlwide on the aid of those affected by mass spread diseases and so it was really important to make it an entertaining experience for the users. It's a one page static site, developed start to finish by me. It features a fully responsive, interactive map, modals for all the diseases listed, carousels, animations and a video modal on the topic. The user can share the site with dinamic links that lead to the specific section of the site that was meant to be shared (even the modals). I also localized the site for UY and CO.

Technologies used:

HTML / SCSS / JavaScript / Responsive design / Swiper.js / jVectorMap / GSAP Animations
